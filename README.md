# Three Phase Step Controlled Inverter

## 📌 Description
This project simulates a three-phase inverter using step signal control in MATLAB/Simulink. The inverter converts DC input into three-phase AC output using controlled switching.

## ⚙️ Features
- Step signal based switching
- Three-phase AC waveform generation
- 120° phase shift between outputs
- Switching sequence implementation

## 🛠️ Tools Used
- MATLAB
- Simulink

## ▶️ How to Run
1. Open the .slx file in MATLAB
2. Run the simulation
3. Observe three-phase outputs in Scope
